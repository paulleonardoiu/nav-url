ABOUT

How to install 'npm'

  1. Go to nodejs.ord
  2. Download recommended LTS version
  3. Open terminal in Administrator Mode
  4. Run command npm --version to verify installation
  5. Create a Visual Studio project
  6. Open Visual Studio terminal and run "npm init"
  7. Load all defaults which can be changed in package.json later

How to install 'live-server'

  1. Open Visual Studio project
  2. Open Terminal within Visual Studio
  3. Run 'npm -g install live-server'
  4. * If command errors then run 'sudo npm install -g npm'
  5. Run 'live-server --version' command to verify installation
  6. Run 'live-server' command to run project on local server
